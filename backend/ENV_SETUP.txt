═══════════════════════════════════════════════════════════════
  .env File Setup Guide
═══════════════════════════════════════════════════════════════

CREATE A FILE NAMED: .env
LOCATION: backend/.env

COPY THIS CONTENT:
─────────────────────────────────────────────────────────────

OPENAI_API_KEY=YOUR_OPENAI_KEY_HERE
OPENAI_MODEL=gpt-4o-mini
ROBLOX_API_KEY=
DATABASE_URL=sqlite:///./vibe_coding.db
CORS_ORIGINS=http://localhost:5173
PORT=8000

─────────────────────────────────────────────────────────────

REQUIRED:
─────────
✅ OPENAI_API_KEY - Get from https://platform.openai.com/api-keys
   Replace: YOUR_OPENAI_KEY_HERE with your actual key

OPTIONAL:
─────────
⚪ ROBLOX_API_KEY - Leave empty for now (optional)
   Will be needed later for Roblox Cloud API integration

⚪ OPENAI_MODEL - Default is "gpt-4" (can use "gpt-3.5-turbo")
⚪ DATABASE_URL - Default works fine
⚪ CORS_ORIGINS - Default works fine
⚪ PORT - Default is 8000

EXAMPLE:
────────
OPENAI_API_KEY=YOUR_OPENAI_KEY_HERE
OPENAI_MODEL=gpt-4o-mini
ROBLOX_API_KEY=
DATABASE_URL=sqlite:///./vibe_coding.db
CORS_ORIGINS=http://localhost:5173
PORT=8000

═══════════════════════════════════════════════════════════════
  That's it! Save the file as .env in the backend folder
