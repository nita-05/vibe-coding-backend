## ‚úÖ Best Prompts for Simple Playable Games

### **1. Coin Collector Game** (Recommended for Beginners)
```
Create a coin collector game with:
- 50 glowing yellow coins scattered randomly across the map
- Red obstacles that kill the player when touched
- Score UI showing coins collected in the top-left corner
- Win message when player collects 25 coins
- Coins respawn after collection
```

### **2. Simple Obby (Obstacle Course)**
```
Create an obby game with:
- 10 floating platforms the player must jump between
- Moving obstacles on some platforms
- Checkpoint system every 3 platforms
- Finish line at the end with celebration message
- Timer showing how long it takes to complete
```

### **3. Racing Game**
```
Create a racing game with:
- 3 laps around a track
- Checkpoints every 100 studs
- Lap counter UI showing current lap
- Finish line that shows final time
- Simple car that moves with WASD keys
```

### **4. Platformer Game**
```
Create a platformer game with:
- Player can jump and move left/right
- 5 levels with increasing difficulty
- Collectible stars in each level
- Enemies that move back and forth
- Level complete message when all stars collected
```

### **5. Survival Game**
```
Create a survival game with:
- Player health bar (100 HP)
- Enemies spawn every 10 seconds
- Player can shoot projectiles to defeat enemies
- Score increases for each enemy defeated
- Game over screen when health reaches 0
```

### **6. Tycoon Game**
```
Create a simple tycoon game with:
- Click button to earn coins
- Upgrade button to increase coins per click
- Auto-earn system that gives coins every 5 seconds
- UI showing total coins and coins per second
- Visual feedback when coins are earned
```

## üí° Prompt Writing Tips

### ‚úÖ **DO:**
- **Be specific**: "50 coins" instead of "some coins"
- **List features clearly**: Use bullet points or numbered lists
- **Mention UI elements**: "Score UI in top-left corner"
- **Describe visuals**: "glowing yellow coins", "red obstacles"
- **Include win conditions**: "Win when player collects 25 coins"

### ‚ùå **DON'T:**
- ‚ùå "Make a game" (too vague)
- ‚ùå "Something fun" (not specific)
- ‚ùå "A complex RPG" (too complicated for first try)
- ‚ùå "Like Fortnite" (too complex, be specific about features)

## üéì Pro Tips

1. **Start Simple**: Generate a basic game first, then add features one by one
2. **Test Incrementally**: After each refinement, test in Roblox Studio
